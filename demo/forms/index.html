Forms

Form-Validation

FlexCss contains a comprehensive form library with a solid JavaScript plugin that utilises HTML5 Form-Validation. FlexCss.Form overwrites the default behaviour of the browser to support a consistent UI across all browsers and adds support for custom validators and inline validation (on blur).

Features

  • Plugin-In replacement for HTML5-Form-Validation
  • Inline-Validation
  • Custom-Validation/Validators
  • Submitting and Validating a form via XHR (json or form encoded)
  • Beautiful validation UI (Tooltip or inline)
  • Event System

Get-Started

Initialize the plugin for all forms on a page with the following line:

{% highlight javascript %} FlexCss.Form.init('form', optionalOptions); {% endhighlight%}

The first param form may be a selector defined by Document.querySelector(). The second one allows to overwrite defaultOptions for each form. All Options may be set trough a data attribute.

{% highlight javascript %} { // if true creates tooltips above element, uses FlexCss Tooltips createTooltips: true, // if true appends error message after input element appendError: false, // type of ajax submit ajaxSubmitType: 'POST', // json content type if ajax method is set to json ajaxJsonContentType: 'application/json; charset=utf-8', // allow inline validation inlineValidation: true } {% endhighlight%}
